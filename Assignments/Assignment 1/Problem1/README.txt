This subfolder of Problem 1 has 3 diffrent parts; p1_a, p1_b and p1_c that corresponds to part a, part b and part c of Problem 1 respectively.

part (a)
	The code is compiled with 
		gcc p1_a.c -o p1_a
	It is run with
		./p1_a
		
part (b)
	The code is compiled with 
		gcc p1_b.c -o p1_b
	It is run with
		./p1_b

part (c)
	The code is compiled with 
		gcc p1_c.c -o p1_c
	It is run with
		./p1_c &
	Followed by 
		ps -l
