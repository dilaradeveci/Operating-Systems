This subfolder of Problem 3 has 2 diffrent parts; p3_a1 and p3_a1 that correspond to Part I and Part II of Part a respectively.
 Makefile is also included in the subfolder.

part (a1)
	The code is compiled using the Makefile accompanying the source and the command "make"
	The modules are loaded using the "sudo insmod p3_a1.ko" command
	The message confirming module loading in the kernel log buffer is checked with the dmesg command
	The modules are removed using the "sudo rmmod p3_a1" command
	The message confirming module removal in the kernel log buffer is checked again with the dmesg command
		
part (a2)
	The same steps as part a1 are followed.
